Canal-web

Canal is a web based cellular automata analysis tool. You can set your initial state and run any of the ECA rules on it. There are more options that extends the possibilities of analysis and simulations.

ERoulette

If you have problems taking decisions ERoulette is the solution. ERoulette is a customizable roulette desktop application that helps you to choose a random alternative. Make your own options image, assign probabilities and spin it!. As a demostration, ERoulette application is preconfigured as a day activity roulette to choose what thing to do someday.

Far Flight

Far Flight is a simple html5 canvas game. Your objective is to fly as far as you can without crashing the blocks in the way. The speed increases through time making the game harder everytime.

Hanoi

Hanoi is a simple Html5 pure canvas Towers of Hanoi game demo. Towers of Hanoi is a puzzle game invented by French mathematician Edouard Lucas in 1883. pitri.js

pitri.js is a small javascript library for projecting points with perspective. It isn't dependant of any framework so you can use it in anything. pitri.js provides you functions to define a camera view and to transform a 3d point to viewport coordinates.

Polyshape

Polyshape is a simple html5 pure canvas 2d projection and rotation demo. You can rotate a shape or see it around in a first person perspective.

Tundra Piechart

Tundra is a piechart viewer and editor software for bioinformatics. You can import data from several sources including MGTaxa, QIIME, Krona and plain text. Edition is very interactive with high degree of freedom. You can export your piechart to PNG and SVG image formats.
